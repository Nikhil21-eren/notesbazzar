# NotesBazaar — Jharkhand का Study Hub

## Deploy in 2 minutes
1. Go to vercel.com → New Project
2. Upload this folder
3. Click Deploy!

## Firebase Rules
Go to Firebase Console → Firestore → Rules
Paste content from firestore.rules → Publish

## Features
- Email + Google login
- Upload notes with Google Drive links
- Download counter
- Save notes
- Filter by college & type
- Search in Hindi + English
- Mobile-first design
